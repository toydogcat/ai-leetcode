## 題目描述
你和你的朋友正在玩 Nim 遊戲。

桌子上有一堆石頭。你們輪流取出 `1` 到 `3` 塊石頭。你先手。
取走最後一塊石頭的人贏得比賽。

假設你們兩人都非常聰明，都會做出最佳決策。請寫一個函數，判斷給定桌子上的石頭數量 `n` 時，你是否能夠贏得比賽。

**範例 1:**
```
輸入: n = 4
輸出: false
解釋: 如果桌子上有 4 塊石頭，那麼你永遠無法贏得比賽。不管你拿 1、2 或 3 塊，最後一塊石頭都會被你的朋友拿走。
```

**範例 2:**
```
輸入: n = 1
輸出: true
```

**範例 3:**
```
輸入: n = 2
輸出: true
```

## 解題心得：數學與博弈論分析
這是一道經典的博弈問題。我們可以藉由歸納法來分析勝負規律：

- 當 $n = 1, 2, 3$：你直接拿走全部石頭，**先手必勝**。
- 當 $n = 4$：你拿走 $x$ 塊（$x \in \{1, 2, 3\}$），剩下 $4 - x$ 塊。對手可以直接把剩下的石頭全部拿走。**先手必敗**。
- 當 $n = 5, 6, 7$：你分別可以拿走 $1, 2, 3$ 塊，讓剩下石頭剛好為 $4$ 塊留給對手。由於對手面對 $4$ 塊必敗，因此你 **先手必勝**。
- 當 $n = 8$：無論你拿多少塊，對手都能把剩下的石頭控制在 $4$ 塊留給你。因此你 **先手必敗**。

**結論**：只要石頭個數是 `4` 的倍數（即 `n % 4 == 0`），先手必定無法獲勝；否則，先手一定能做出最佳策略獲勝。

- **時間複雜度**: $O(1)$。
- **空間複雜度**: $O(1)$。

## 程式碼實作

### Python
```python
class Solution:
    def canWinNim(self, n: int) -> bool:
        return n % 4 != 0
```

### C++
```cpp
class Solution {
public:
    bool canWinNim(int n) {
        return n % 4 != 0;
    }
};
```
