## 題目描述
給你一個由 `n` 個整數組成的數組 `nums` ，和一個目標值 `target` 。請你找出並返回滿足下述全部條件且不重複的四元組 `[nums[a], nums[b], nums[c], nums[d]]` 。

## 解題心得：多一層套娃
這題是 15 題（三數之和）的進階版。如果你會三數之和，這題只是多穿一層衣服而已。

**核心邏輯：**
1. **排序**：同樣先排好序，方便去重和控制大小。
2. **第一層循環**：固定第一個數 `i`。
3. **第二層循環**：固定第二個數 `j`。
4. **雙指針**：剩下的兩個數就變成了一個「兩數之和」問題，用左右指針 `left` 和 `right` 夾逼。
5. **關鍵是去重**：不論是 `i`, `j` 還是雙指針，遇到跟剛才一樣的數字都要跳過，否則答案會重複。

這就像是剝洋蔥，外面那一層是四數之和，剝開變三數，再剝開變兩數。雖然層數變多了，但內核的原理都是一樣的。

### Python
```python
class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        res = []
        n = len(nums)
        for i in range(n - 3):
            if i > 0 and nums[i] == nums[i-1]: continue
            for j in range(i + 1, n - 2):
                if j > i + 1 and nums[j] == nums[j-1]: continue
                l, r = j + 1, n - 1
                while l < r:
                    s = nums[i] + nums[j] + nums[l] + nums[r]
                    if s < target: l += 1
                    elif s > target: r -= 1
                    else:
                        res.append([nums[i], nums[j], nums[l], nums[r]])
                        while l < r and nums[l] == nums[l+1]: l += 1
                        while l < r and nums[r] == nums[r-1]: r -= 1
                        l += 1; r -= 1
        return res
```

### C++
```cpp
#include <vector>
#include <algorithm>

class Solution {
public:
    std::vector<std::vector<int>> fourSum(std::vector<int>& nums, int target) {
        std::sort(nums.begin(), nums.end());
        std::vector<std::vector<int>> res;
        int n = nums.size();
        for (int i = 0; i < n - 3; i++) {
            if (i > 0 && nums[i] == nums[i-1]) continue;
            for (int j = i + 1; j < n - 2; j++) {
                if (j > i + 1 && nums[j] == nums[j-1]) continue;
                int l = j + 1, r = n - 1;
                while (l < r) {
                    long long s = (long long)nums[i] + nums[j] + nums[l] + nums[r];
                    if (s < target) l++;
                    else if (s > target) r--;
                    else {
                        res.push_back({nums[i], nums[j], nums[l], nums[r]});
                        while (l < r && nums[l] == nums[l+1]) l++;
                        while (l < r && nums[r] == nums[r-1]) r--;
                        l++; r--;
                    }
                }
            }
        }
        return res;
    }
};
```
