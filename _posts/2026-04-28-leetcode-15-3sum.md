## 題目描述
給你一個包含 `n` 個整數的數組 `nums`，判斷 `nums` 中是否存在三個元素 `a, b, c` ，使得 `a + b + c = 0` ？請你找出所有和為 `0` 且不重複的三元組。

## 解題心得：從混亂到秩序
這是面試中最愛考的「雙指針」題目之一。

**核心邏輯：**
1. **先排好序**：這是靈魂。排好序後，我們才知道數字的大小分布，也才能避免重複的答案。
2. **固定一個，找另外兩個**：遍歷數組，把當前的數 `nums[i]` 固定住。我們現在的任務就是找另外兩個數，讓它們的和等於 `-nums[i]`。
3. **雙指針左右包抄**：
   - 既然數組是排好序的，我們用 `left`（左邊最小）和 `right`（右邊最大）兩頭往中間縮。
   - 如果三數之和太大，就把右邊往左移。
   - 如果太小，就把左邊往右移。
4. **絕對不吃回頭草**：不論是固定的數字還是移動的指針，遇到跟剛才一模一樣的數字都要直接跳過（去重）。

這就像是在相親大會上配對。我們先按身高排好隊。固定一個人後，剩下的兩個人要一高一矮來搭配，如果這組試過了，下次遇到身高一樣的人就直接跳過，省時省力。

- **時間複雜度**: $O(n^2)$
- **空間複雜度**: $O(\log n)$ (排序所需的空間)

## 程式碼實作

### Python
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = []
        for i in range(len(nums) - 2):
            if i > 0 and nums[i] == nums[i-1]:
                continue
            l, r = i + 1, len(nums) - 1
            while l < r:
                s = nums[i] + nums[l] + nums[r]
                if s < 0:
                    l += 1
                elif s > 0:
                    r -= 1
                else:
                    res.append([nums[i], nums[l], nums[r]])
                    while l < r and nums[l] == nums[l+1]: l += 1
                    while l < r and nums[r] == nums[r-1]: r -= 1
                    l += 1
                    r -= 1
        return res
```

### C++
```cpp
#include <vector>
#include <algorithm>

class Solution {
public:
    std::vector<std::vector<int>> threeSum(std::vector<int>& nums) {
        std::sort(nums.begin(), nums.end());
        std::vector<std::vector<int>> res;
        for (int i = 0; i < (int)nums.size() - 2; i++) {
            if (i > 0 && nums[i] == nums[i-1]) continue;
            int l = i + 1, r = nums.size() - 1;
            while (l < r) {
                int s = nums[i] + nums[l] + nums[r];
                if (s < 0) l++;
                else if (s > 0) r--;
                else {
                    res.push_back({nums[i], nums[l], nums[r]});
                    while (l < r && nums[l] == nums[l+1]) l++;
                    while (l < r && nums[r] == nums[r-1]) r--;
                    l++; r--;
                }
            }
        }
        return res;
    }
};
```
